## 联合类型（|）

- 一般用于基本类型定义
    ```js
    let x: number | string = "aaa";
    let y: number | string = 123;
    ```
    - 对象既可以是字符串类型，也可以是数字类型
    - 取的是两个的并集，两个中任一个都可以

## 交叉类型（&）

- 一般用于interface
    ```js
    interface Man {
        run(): void;
    }
    
    interface Worker {
        work(): void;
    }

    interface maleWorker: Man & Worker = {
        run() { },
        work() { }
    }
    ```
    - 取的是两个的合并，两个都必须要有
